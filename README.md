# isaac-bot

A Discord bot for *The Binding of Isaac* built with discord.js and TypeScript.

## Commands

- **Item Lookup** — `/isaac lookup <item>` — Look up items from The Binding of Isaac
- **Spindown Dice Checker** — `/isaac spindown <item>` — Shows the next 10 items in the spindown dice sequence

## Setup

1. Copy `.env.example` to `.env` and fill in your Discord bot token and client ID
2. Run `npm install`
3. Run `npm run dev` for development or `npm run build && npm start` for production
