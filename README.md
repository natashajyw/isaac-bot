# isaac-bot

A Discord bot for *The Binding of Isaac* built with discord.js and TypeScript.

## Commands

- **Item Lookup** — `/isaac lookup <item>` — Look up items from The Binding of Isaac
- **Spindown** — `/isaac sp <item>` — Shows the next 10 items in the spindown dice sequence (descending by ID)
- **Spindown Find** — `/isaac spfind <item>` — Find items that spin down into the given item (next 10 IDs above)
